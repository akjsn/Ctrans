Shape Of Numbers

Fall 2018: Question 22 (use Python to make this program)
Fall 2017: Question 22 (use C language to make this program)

Write a program to output shapes using numbers.

Input instructions:

  1. Input two numbers every time.
  2. First, input a number. This number will determine how the output will look like: 1 for diamond, 2 for triangle (pointing down).
  3. Next, input a second number which will determine the number of rows the shape will span.
    • If the first input is a 1, this number has to be an odd number from 1 to 18.
    • If the first input is a 2, this number has to be an odd number from 1 to 5.
  4. An input of -1 will terminate the program.

Output instructions:

  1. For diamonds, output the shape with 1 at the edges.
    For triangles, output the shape with 1 in the center.
    Example outputs are shown below.
  2. For any invalid input, output E.
  3. Please leave an empty line after every output is completed.
