n Words Before and After...

Spring 2019: Question 12 (use C language to write this program)
Spring 2017: Question 11 (use C language to write this program)

The objective of this program is to output n words before and after a given word in a phrase.

Input instructions:

  1. First, input the following in order: word, number of words, and arrangement direction (L or R; L for left, R for right)
  2. Next, input a phrase with maximum length of 5 lines (in 5 separate lines). If you do not want to input a phrase in a line, input 0 for that line.
    Note that the phrases can only contain letters and spaces. The phrases are not case sensitive.

Output instructions:

  1. For every appearance of the given word in the first line of input, output that word as well the n words before and after it.
    Output the given word in uppercase, and everything else in lowercase.
  2. Arrange them on increasing number of letters before or after the word, based on the arrangement direction given in the input. If the number of letters are the same, arrange them in increasing alphabetical order.
