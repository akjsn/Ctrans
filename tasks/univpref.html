University Preferences

Spring 2019: Question 30 (use C language to make this program)

A final year high school student is deciding which university would he/she go for further studies. There are many universities in the area, and each university has its own features, as shown:

  • BC: Big Campus
  • NC: Next to City -- university is located close to a big city.
  • CT: Convenient Transportation
  • NS: Next to Sea -- university is located by the sea.
  • NM: Next to Mountain -- university is located in the mountainside.
  • HL: Has Lake -- the university features a lake.
  • NL: Near Landscape -- university is located near a scenic area.
Among the features, the user can define his/her desired features. If more than one feature set is desired, then + can be used to define an OR relationship.
For example: BC NS + CT HL means that we are looking for universities which have a big campus and are located by the sea, or universities with convenient transportation and a lake.

You must define University as a structure or class.

Input instructions:

  1. First, input the number of universities n (not exceeding 10).
  2. For the next n lines, input a string of words, with the first word being the name of the university (no more than 10 characters), and the remaining words being the university's features (two characters each).
  3. Then, input the number of queries to be done m (not exceeding 10).
  4. Then, for the next m lines, input the search criteria by providing the university's feature(s).
    Features can be separated with or without a space.
    The presence of a + defines an OR relationship, otherwise an AND relationship will be defined.
    XXYY+AABB
    XX YY + AA BB
    Both of the above inputs define a query for universities with features XX and YY, or features AA and BB.

Output instructions:

  1. Output m lines, with each line containing the names of the university(ies) which match the criteria defined in input.
  2. If more than one university satisfies the criteria defined in input, output the names of those universities in dictionary order, separated with a space.

Sample Input Expected Output
5
NSYSU NC CT NS NM
NTU BC NC CT NS
NCCU BC NL HL
Providence BC NC
NTHU BC NS
2
BC NS + CT HL
NM + BC NL + BC NC 		NTHU NTU
