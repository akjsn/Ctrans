Largest Prime Number of k

Fall 2018: Question 29 (use Python to write this program)
Fall 2017: Question 29 (use C language to write this program)

In this program, the user inputs a number x, and using the following rules, another number k is decided.

If x is... ...then k will be... Example (x → k)
a positive integer from 2 to 1000 x 50 → 50
0 or 1 1000 0 → 1000
a negative integer x * -11 -90 → 990
a positive floating number the number before the decimal point 66.22 → 66
a negative floating number the number after the decimal point -33.44 → 44

Write a program to compute the largest prime number in the range of 2 to k.

Input instruction: Input a number.

Output instructions:

  1. Output the largest prime number in the range of 2 to k.
  2. Output "Error!" if the input is invalid.

Sample Input Expected Output
50 47
0 997
90 983
66.22 61
-33.44 43
1001 Error!
