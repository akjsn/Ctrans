Spring 2017: Question 13 (use C language to write this program)

Write a program to simulate a digital circuit. Input is an 8-bit binary number (00000000 (0) to 11111111 (255) in range), and output will be in the form of an 11-bit binary number.

The digital circuit has three components:

  1. Loop circuit
    The loop circuit does operations on a decimal number m as follows:
    • C(m) = m if m = 0 or m = 1
    • C(m) = C(m / 2) if m is even
    • C(m) = C((m + 1) / 2) if m is odd
  2. Counter
    The counter tracks how many times the number runs through the loop circuit.
    For example: m=00001010 (decimal 10), the loop circuit will return 5, 3, 2, and 1 in its iterations. In total, the loop circuit is run 4 times.
    Note that this output should be converted to 4-bit binary; therefore from the above example, the output will be 0100.

Input instructions:

  1. Input an 8-bit binary number.
  2. After that, input 0 to continue input (go back to Step 1), or input -1 to end input.

Output instruction: Output the 4-bit binary number result.
