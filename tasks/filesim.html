File Simulator

Spring 2019: Question 23 (use C language to make this program)

Write a program to simulate file operations.

The program has the following functions:

  1. Write File
    Refer to the following code for a guide on the file structure:
    struct file {
    char* name;
    int list[];
    };
  2. Read File
  3. Edit File
  4. Exit

Instructions: Each function has its own set of instructions.

  • Write File
    1. First, input a file name. (name)
    2. Then, input numbers from 0 to 255 one number at a time. Every number is stored sequentially in the file (we are using list for this one).
    3. Input -1 to end input.
    4. If the file names repeat, output "Error".
  • Read File
    1. First, Input the file name. If the file does not exist, output "Error".
    2. Output the file size in bytes.
    3. Output the file's contents in binary, with each bit separated.
  • Edit File
    1. First, Input the file name. If the file does not exist, output "Error".
    2. Input a number m, which represents the item to be edited.
    3. Input a second number n, which represents the new value for m.
