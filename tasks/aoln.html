Arithmetic Operations on Long Numbers

Fall 2018: Question 47 (use C language to write this program)
Fall 2017: Question 37 (use C language to write this program)

Design a program to perform arithmetic operation on two long numbers (each with no more than 60 digits), and calculate the addition and subtraction of those numbers.

Input instruction: Input two numbers, each not exceeding 60 digits.

Output instruction: Output the addition and subtraction result of the two numbers.

Arrays are required in this exercise.

For the sake of clarity, numbers in the below test cases include spaces every 3 digits.
DURING ACTUAL TESTING, INPUT THE NUMBERS WITHOUT SPACES IN BETWEEN.
DO NOT INCLUDE SPACES IN THE ACTUAL OUTPUT OF THE PROGRAM.
