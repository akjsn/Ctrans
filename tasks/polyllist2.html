More Polynomials Using Linked Lists

Spring 2018: Question 36 (use C language to write this program)

Using a linked list, input a polynomial, and then input the number of addition or subtraction operations to be done as well as the amount to be added/subtracted.

Input instructions:

  1. First, input n numbers, which represent coefficients, with the first number representing a variable (x) raised to the power of n-1, the second number representing x raised to the power of n-2, and so on until the last number, which is simply x raised to the power of 0.
  2. Next, input a second number p. This number represents the number of addition or subtraction operations.
  3. From the third line onward, input p lines of data, with each line representing an addition or subtraction operation. Input the power, followed by the coefficient.

Output instructions:

  1. Output the resulting polynomial after the operations.
  2. If the resulting variable has a coefficient of 0, the variable cannot be displayed.
  3. If the resulting polynomial is 0, output 0.
