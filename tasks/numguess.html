Guess A Number

Fall 2018: Question 28 (use Python to write this program)
Fall 2017: Question 41 (use C language to write this program)

In this program, a four-digit number X using numbers from 0 to 9 is preset. The numbers do not repeat.

A user inputs a four-digit number as well as the similarity of the number to X, and this four-digit number is compared with X.

The process to determine similarities will be as follows:

  1. If there is a number that matches the number in the answer, and is also in the same position as the position of the number in the answer, note this down as nA (n being the amount of numbers that satisfy this rule).
    For example: if X is 1234, and the user guesses 1856, record this as 1A.
  2. If there is a number that matches the number in the answer, but not in the same position as the position of the number in the answer, note this down as nB (n being the amount of numbers that satisfy this rule).
    For example: if X is 1234, and the user guesses 8561, record this as 1B.
  3. Rule (1) takes priority over Rule (2).
  4. Input will be given until X is guessed.

Input instruction: For every line, input a four-digit number guess, as well as the similarity of that number to X.

Output instruction: Output the correct answer X.
