Sum of Consecutive Integers

Fall 2018: Question 26 (use Python to write this program)
Fall 2017: Question 26 (use C language to write this program)

Write a program to express a given number as a sum of consecutive integers.

For example, 9 can be expressed as 2 + 3 + 4, or 4 + 5, or 9.

Input instruction: Input a positive integer N.

Output instructions:

  1. Output all the ways to add consecutive integers to reach the result N.
  2. Output the ways with the smallest starting number at the top.
  3. Output E if the input is invalid.

Sample Input Expected Output
9 2+3+4
4+5
9
-1 E
A E
