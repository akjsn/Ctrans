Average Course Grade

Spring 2018: Question 10 (use Python to write this program)

Xiao Ming takes various courses in his first two years of university. He wants to know, for each professor, his average score for all the courses he took that are taught by that professor, in order to find out whose professor’s course should he take in the next semester.

Input instructions:

  1. First, input two numbers N and M.
    N denotes the number of courses taken; and M denotes the number of professors.
    N should be a number between 2 and 100.
    M should be a number between 0 and 10.
  2. Next, input N lines of data, which contain the following in order:
    Course ID, Course name, and the final grade.
    Please store this information in a dictionary.
  3. Then, input M lines of data, which contain the following in order:
    Professor’s ID, Professor’s name, and the courses taught.
    Please store this information in a list.
  4. Lastly, input the professor’s name.
  5. For all inputs from (1) to (3), separate each piece of information with a space.

Output instructions:

  1. Output the average score (in integer).
  2. If the inputs are out of range, output “Error”.
  3. If the professor’s name is not in the list, output “查無此人” (person not found).

Sample Input (note: inputs here are in Chinese) Expected Output
5 3
232100 計算機程式設計(一) 90
231543 微算機系統 75
238794 計算機程式設計(二) 96
238793 離散數學 65
238796 計算機組織 60
001 郭忠義 232100 238794
002 陳彥霖 231543 238796
003 謝東儒 238793
郭忠義 		93
