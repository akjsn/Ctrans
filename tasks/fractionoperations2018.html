Fall 2018: Question 24 (use Python to write this program)

Write a function to add, subtract, multiply, and divide two fractions.

def execute(n1, d1, n2, d2):

n1: Numerator part of the first fraction
d1: Denominator part of the first fraction
n2: Numerator part of the second fraction
d2: Denominator part of the second fraction
numerator: Numerator part of the addition result
denominator: Denominator part of the addition result

Input instruction: Input two fractions, one in each line.

Output instructions:

  1. Output the addition, subtraction, multiplication, and division result of the two fraction,; one in each line.
  2. If any of the inputs is 0, output “ERROR”.
  3. If the result is greater than 1, then the fraction part has to be in brackets.
    For example: If the result is 1 1/6, output the result as 1(1/6).
  4. If the result is negative, place the negative sign at the front.
  5. If the result is 0, display “0”.
  6. All results must be displayed in the simplest form.
    Allowed: 1/2, 2/3, 4/7 (these fractions are in their simplest forms)
    Not allowed: 2/4, 4/6, 10/12 (these fractions are NOT in their simplest forms)

If done in C, pointers must be used. Do not use arrays.
Sample Input Expected Output
1/2
2/3 		1(1/6)
-1/6
1/3
3/4
0/2
2/3 		ERROR
ERROR
ERROR
ERROR
