Xiao Ming has a lot of fill-in-the-blanks mathematics homework assignments. The instruction of these assignments is that Xiao Ming has to fill in the fifth number in the sequence based on the previous four numbers in the series. Xiao Ming knows that the questions are either arithmetic series or geometric series, so he decides to make a program to help him finish his homework.

Input instructions:

  1. First, input the number of questions T (1 <= T <= 20).
  2. For every line onward, based on the number of questions, input the sequence of four numbers (all integers).

Output instructions:

  1. For every line onward, based on the number of questions, input the sequence of four numbers (all integers).
  2. For any invalid input, output E.

Sample Input Expected Output
2
1 2 3 4
32 16 8 4 		1 2 3 4 5
32 16 8 4 2
