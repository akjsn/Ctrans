Binary Tree of Numbers

Spring 2019: Question 32 (use C language to make this program)

Write a program to build a binary tree of numbers.

For a value n, the left side of the binary tree has a value less than or equal to n, while the right side of the binary tree has a value larger than n.

Possible functions for this program are shown below:

void Insert(nodep_t *node, student_t key)
{
    // inserts data into binary tree
}

void Inorder(nodep_t *node);
{
    // inorder traversal, traverses the binary tree from left to right
}

Input instructions:

  1. Input i to insert a data into the binary tree.
  2. Input p to traverse the binary tree.
  3. Input e to exit the program.

Output instruction: Whenever p is input,

  1. If the binary tree does not exist, output "null".
  2. The first number input is the root.
  3. Numbers smaller than or equal to the root are placed to the left side of the root.
  4. Numbers bigger than to the root are placed to the right side of the root.
  5. Separate each number in the output with a space.
