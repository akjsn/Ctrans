Number Conversion

Fall 2018: Question 20 (use Python to write this program)
Fall 2017: Question 19 (use C language to write this program)

Build a number conversion system to convert a number in decimal (base 10) into a number in base b.

Input instructions:

  1. First, input the base number b (from 2 to 16).
  2. Then, input the decimal/base 10 number n (from 0 to 1,000,000,000).

Output instructions:

  1. Output the resulting number after conversion.
  2. If the input is invalid, output E.

You have to use while loop and function call in this exercise.
Sample Input Expected Output
16 1234 4D2
8 5645646 327272410
11 17489465 9966104
4 17489 10101101
