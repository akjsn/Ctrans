Sudoku Solver

Spring 2018: Question 22 (use Python to make this program)

Moving on from the earlier Sudoku Checker exercise, we have the same 9-by-9 Sudoku grid, except that 0, which indicates a blank space, is now also included in the grid.

Write a program to find out what number should be filled in the space with a 0, in order to solve the Sudoku puzzle. Remember, the same rules as described in the Sudoku Checker exercise also apply here (except for the 0s).

Input instruction: Input a 9-by-9 grid of numbers from 0 to 9, from which blank spaces are indicated with a 0.

Output instruction: From top to bottom, and left to right, output the number to be filled in the space with a 0.

Sample Input Expected Output
023456789
456789123
789123456
231564897
564897231
897231564
312605978
645978312
908312645 		0 0 1
6 4 4
8 1 7
