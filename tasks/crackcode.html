Cracking The Code

Spring 2018: Question 19 (use Python language to write this program)

Mr. Chang, a rich landowner, stores all his valuables in a safe. The safe is equipped with a password lock. Mr. Chang writes the clues to open the safe on a scroll, and he decides to inherit his wealth to his only son. The contents of the scroll are two lines of sentences, made of alphabets, numbers, spaces, and punctuation marks. The instructions for cracking the code are as follows:

  1. The first sentence contains n numbers. Take the numbers from the sentences and align them to the left. Add up all the digits in the same column, and concatenate the results to form a new number. Do note that:

    • The resulting number must be 4 digits long.
      If the number has more than 4 digits, then take the last 4 digits.
      If the number has less than 4 digits, then add 0 to the missing spaces in the beginning.
  2. For the second sentence, count the appearance of every letter in that sentence. Take the frequency of 4 letters with the most number of appearances (but not exceeding 9), and arrange them based on where the letter is first seen to form another 4-digit number.
    Note that UPPERCASE CHARACTERS and lowercase characters are considered different characters.

Input instruction: Input two sentences, which contain alphabets, numbers, spaces, and punctuation marks.

Output instruction: Output the 8-digit password in one line.

Today is my 45 birthday, There are 65 guest coming for the 1000 party, I got 769 gifts to open.

Processing:
First sentence: Only the 2 columns on the left are counted, and after concatenating the two numbers, we have 1816 as the first set of 4 digits.

Second sentence: From the four letters with the most appearances, we have 7768 as the second set of four digits.
