Spring 2018: Question 12 (use Python to write this program)

Nowadays, you can make friends online. There are people who use this opportunity to expand their circle of friends, just like collecting stamps. People tho have interest in this seem to be enjoying what they do.

Assume that a friendship is a two-way relationship; which means that, if A befriends B, then B will also befriend A.

Input instructions:

  1. First, insert the number of test cases.
  2. For every test case, start by inputting the number of relationships, denoted by F. The number of relationships should not exceed 100,000.
  3. Then, insert F lines of data, with each line containing two names, indicating the formation of a friendship between the two names. Each name should be 1-20 characters long. Separate each name with a space.
    You can store this information in a list or in a dictionary.

Output instruction: Based on the inputs above, output the number of persons in the circle of friends.
