Queue Using Linked Lists

Note: This exercise uses just one queue. For multiple queues, click here.
Spring 2018: Question 38 (use C language to make this program)

Make a program to demonstrate the concept of a queue using linked lists.

Just like a stack, a queue also has two basic functions, but with different usage methods:
push: inserts an item to the end (tail) of the queue
pop: removes an item from the start of the queue

Input instruction: The following commands are used in this program:

  1. push: input a line of data which contains a person’s name, age, birthday (year, month, day, with each number separated by a space). Separate each piece of data with a space.
  2. pop: removes the data from the start of the queue.
  3. This command ends the program.

Output instructions:

  1. When inputting the pop command, input another number, which commands the system to print the following from the data which is removed from the queue:
    1: name
    2: age
    3: birthday (year, month, date). Separate each number with an underscore (_).
  2. If the queue is empty when the pop command is input, output “The stack is empty”. (Translator’s note: it’s supposed to be “The queue is empty...)
