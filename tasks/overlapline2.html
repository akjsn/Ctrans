Overlapping Lines II

Fall 2018: Question 21 (use Python to make this program)
Spring 2018: Question 5 (use Python to make this program)
Fall 2017: Question 20 (use C language to make this program)

Moving on from the previous Overlapping Lines exercise, given a number of line sections, write a program to compute the total spanning lengths of these line sections.

If there are overlaps in the lines, the overlapping sections are counted only once.

You must use while loop and function call in this exercise.
Arrays are not used in this exercise.

Input instructions:

  1. First, input an integer n. This number represents the number of test cases.
  2. For every test case in n, input an integer m. This number represents the number of line sections.
  3. For every line section in m, input two numbers x and y which will represent the start and end points of the line section.
  4. x and y must be numbers from 0 to 60,000, and every line section cannot exceed 5,000 in length.
  5. There should be some space between the end of a line section and the start of another line section.

Output instructions:

  1. Based on the above inputs, display the total spanned length for every test case. For every test case, display this result in a new line.
  2. For any invalid input, output E.
