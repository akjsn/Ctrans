Student Record Query

Spring 2018: Question 35 (use C language to make this program)

Given a student record containing a number of student data, make a program to search and display the student data.

First, input the student data, then input a command to perform different operations on the student record.

Different input formats are used for different commands.

A student data consists of name, ID, and gender. For gender, M represents male, and F represents female.

The commands used in this program are:

  • f: search for student data in the student record. Input the ID after inputting the command.
  • l: output all the student data\
  • i: inserts a new student data into the student record. Input the name, ID, and gender after inputting the command.
  • d: deletes a student data from the student record. Input the ID after inputting the command.
  • m: modifies a student data in the record. Input the ID, followed by the new data (name, ID, gender) after inputting the command.
  • q: exits the program.

Input instruction: The following commands are used in this program:

  1. First, input the number of student data N (1 <= N <= 10).
  2. Next, input N lines of student data, with each line containing the student’s name, ID, and gender. Separate each piece of information with a space.
  3. Input a command, followed by the extra information if neccessary. See the command list above for more details. If extra information has to be input, separate each piece of information (including the command) with a space.

Output instructions:

  1. For the commands f and l, output the name, ID, and gender in the student data, separated with a space.
  2. For any invalid input, output “error".

Lucas 001 M
Kuo 002 M
