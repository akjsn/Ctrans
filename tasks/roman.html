Roman Numerals

Fall 2018: Question 15 (use Python to write this program)

There are seven kinds of letters used in Roman numerals:

  • I for 1
  • V for 5
  • X for 10
  • L for 50
  • C for 100
  • D for 500
  • M for 1000

Use the following rules to determine the correct Roman numeral for an integer:

  1. Note that there is no 0 in Roman numerals.
  2. Roman numerals have no relationship with positional notation numbers.
    It is generally said that Roman numerals are only used to note down numbers, and Roman numerals cannot be used in calculations.
  3. Repeating numbers: If a Roman numeral is repeated, then its value in Arabic numerals is the value of the Roman numeral multiplied by the amount of repetitions.
    e.g. II = 2, III = 3, XX= 20, XXX=30
  4. Subtraction rule: If a Roman numeral with a smaller value is written before a Roman numeral with a bigger value, its value in Arabic numerals is the value of the bigger Roman numeral minus the value of the smaller Roman numeral.
    e.g. IV = 4, IX = 9
    • Only I, X, and C can be used in the subtraction rule; therefore, 45 cannot be written as VL (correct form is XLV)
    • The subtraction rule cannot be used to subtract numbers which are separated more than one position away, for example:
      99 cannot be written as IC (correct form is XCIX)
      Explanation: 99 is written as XCIX, because it is represented like this: [ ( 100 – 10 ) + ( 10 – 1 ) ]
    • The subtraction rule can only be used for numbers with a difference of 1 or a difference which is a power of 10 from the next tier, for example:
      • IX (for 9) is allowed, however, IIX (for 8) is not allowed.
      • XC (for 90) is allowed, however, XXC (for 80) is not allowed.
  5. Addition rule: If a Roman numeral with a bigger value is written before a Roman numeral with a smaller value, its value in Arabic numerals is the value of the bigger Roman numeral plus the value of the smaller Roman numeral. For example:
    VI = 6, XI = 11
    • The addition rule cannot be used above three numbers. For example:
      14 is XIV, not XIIII (see Rule #6)
  6. Limitations: The same Roman numeral can only appear a maximum of three times. For example:
    40 cannot be written as XXXX (correct form is XL)
