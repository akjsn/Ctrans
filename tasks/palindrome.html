Palindromic Numbers

Fall 2018: Question 27 (use Python to write this program)
Fall 2017: Question 24 (use C language to write this program)

Write a program to list out all palindromic numbers between two input numbers.

For example:

  • An input of 80 and 100 will give 88 and 99 as the result.
  • An input of 100 and 200 will give 101, 111, 121, 131, 141, 151, 161, 171, 181, and 191 as the result.

Input instruction: Input two numbers.

Output instructions:

  1. List all the palindromic numbers in the range of the two input numbers.
  2. If there are no palindromic numbers in the range of the two input numbers, output 0.

Sample Input Expected Output
1 10 0
10 70 11 22 33 44 55 66
1300 2000 1331 1441 1551 1661 1771 1881 1991
17000 18000 17071 17171 17271 17371 17471 17571 17671 17771 17871 17971
