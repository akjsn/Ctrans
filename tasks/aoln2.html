Arithmetic Operations on Long Numbers... Again

Spring 2019: Question 3 (use C language to write this program)

Design a program to perform arithmetic operation on two long numbers (each with no more than 60 digits), and calculate the addition, subtraction, and multiplication result of these numbers.

Input instruction: Input two numbers, each not exceeding 60 digits.

Output instruction: Output the addition, subtraction, and multiplication result of the two numbers.

Arrays are required in this exercise.

For the sake of clarity, numbers for the multiplication result in the test case below include spaces every 3 digits.
DO NOT INCLUDE SPACES IN THE ACTUAL MULTIPLICATION OUTPUT OF THE PROGRAM.
Sample Input Expected Output
186978448
484312244 		671290692
-297333796
90 555 951 730 517 312
