Sum of Factorials

For the 2018 version of this exercise, click here.
Spring 2018: Question 1 (use Python to write this program)
Fall 2017: Question 17 (use C language to write this program)

Write a program to display the sum of the factorials from the prime factors of an input number.

Input instruction: Input an integer greater than 3.

Output instructions:

  1. Output the factorial from every prime factor less than the input number.
  2. Display the sum of all the factorials in (1).
  3. If the input is invalid (decimals, negative numbers, letters), output “error”.

Sample Input Expected Output
10 2 120
122
10.2 error
