Fall 2018: Question 35 (use Python to write this program)

There are four suits in a standard deck of poker cards: spades, hearts, diamonds, and clubs.
In this exercise, these suits are defined as S, H, D, and C, respectively.
Each suit contains 13 cards from 1 to 13, for a total of 52 cards in four suits.
In this exercise, the values of the cards are from 2 to 13. An Ace (A) has a value of 14.

Rank of suits: Spade > Heart > Diamond > Club

In this program, each card has a code in this format: number and suit.
For example, 10S represents the 10 in spades, 7D represents the 7 in diamonds, and 12C represents the Queen of clubs.

Write a program to determine the rank of a hand of cards in a poker game.

Input instruction: Input the ID of five cards, separated with a space.

Output instruction: Output a number between 0 and 7, using the following rules:

  1. If there are two cards of the same rank, the player has a Pair. Output 1.
  2. If there are two different Pairs in a hand of cards, the player has a Two Pair. Output 2.
  3. If there are three cards of the same rank, the player has a Three of a Kind. Output 3.
  4. If the five cards' ranks are in a sequence (e.g. K, Q, J, 10, 9), the player has a Straight. Output 4.
    (Note: 13 (K), 14 (A), 2, 3, 4 is also considered a Straight in this exercise, but not in the actual game! )
  5. If there is a Three of a Kind and a Pair in a hand of cards, the player has a Full House. Output 5.
  6. If there are four cards of the same rank, the player has a Four of a Kind. Output 6.
  7. If all five cards are in a sequence and are of the same suit, the player has a Straight Flush. Output 7.
  8. If none of the above apply, the player has a High Card. Output 0.

Sample Input Expected Output
9D 8C 8S 8H 9S 5
