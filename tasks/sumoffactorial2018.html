For the older version of this exercise, click here.
Fall 2018: Question 40 (use C language to write this program)
Fall 2018: Question 18 (use Python to write this program)

Write a program to display the sum of the factorials from the prime factors of an input number.

Input instruction: Input an integer between 0 and 1000.

Output instructions:

  1. Output the every prime factor less than the input number and its factorial, separated with a comma.
  2. Display the sum of all the factorials in (1).
  3. If the input is invalid (decimals, negative numbers, letters), output “E”.

Sample Input Expected Output
10 2,2
5,120
122
12 2,2
3,6
8
30 2,2
3,6
5,120
128
-1 E
10.2 E
11.a2 E
