Checking the Report Card

Spring 2019: Question 5 (use C language to write this program)

Design a program qhich allows the query of the grades of a number of students. Note that the program will allow queries of students' report cards repeatedly.

Input instructions:

  1. First, input the number of students.
  2. For each student:
    • First, input the ID and name.
    • Then, input the number of subjects.
    • For each subject, input the subject name, credits, and the grade of that subject.
  3. To query a student's report card record, simply input the ID.
  4. Input 0 to end the program.

Output instructions:

  1. For each query result:
    • First, output the ID and name.
    • Next, output the subject name, credits, and grade for the subject -- one line at a time.
    • Then, output the following in order, each in a new line: credits earned (a student gets the credits for the subject if the student gets a grade of at least 60 for that subject), total average (rounded up to 2 decimal places), rank (determined based on total average)
  2. If there is no data for the input student ID during query, output "ERROR".
