Largest Area and Perimeter

Fall 2018: Question 43 (use C language to write this program)
Fall 2017: Question 31 (use C language to write this program)

Write a program to compute the largest area and perimeter based on a series of data inputs in the form of coordinates on a cartesian plane.

void compute(int x1, int y1, int x2, int y2, int *perimeter, int *area);

(x1, y1): First point
(x2, y2): Second point
*perimeter : Perimeter result
*area: Area result

Input instructions:

  1. For every line, input two points the following order: x1, y1, x2, y2.
  2. An input of -1 ends all input.
  3. Please use absolute value when calculating the lengths of each side of the rectangle.

Output instructions:

  1. Output the largest area and perimeter from all the given inputs.
  2. If the given input cannot form a rectangle, output “Error” and terminate the program.

If done in C, pointers must be used. Do not use arrays.
Sample Input Expected Output
-10 -10 20 60
2 2 52 52
85 8 5 38
-1 		2500
220

Good Luck!
