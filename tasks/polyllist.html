Polynomials Using Linked Lists

Spring 2018: Question 33 (use C language to write this program)

Use linked lists to store the data of two polynomials, then find out the product of the two polynomials.

For example, if we have these two polynomials:
2x4 + 3x3 + x - 1, and
x5 - x4 + 4x2 - 3x + 2,

then the product of the two polynomials will be:
2x9 + 3x8 - 2x7 + 6x6 + 5x5 - 6x4 + 11x3 - 7x2 + 5x - 2

Input instructions:

  1. Input two lines of data, with each line representing a polynomial.
  2. For each line, input n numbers, which represent coefficients, with the first number representing a variable (x) raised to the power of n-1, the second number representing x raised to the power of n-2, and so on until the last number, which is simply x raised to the power of 0.

Output instructions:

  1. Output the product of the two polynomials.
  2. If the resulting variable has a coefficient of 0, the variable still has to be displayed.

Sample Input Expected Output
2 3 0 1 -1
1 0 -1 4 -3 2 		2 3 -2 6 5 -6 11 -7 5 -2
