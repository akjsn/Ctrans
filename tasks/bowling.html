Bowling

Fall 2018: Question 13 (use Python to make this program)
Fall 2017: Question 13 (use C language to make this program)

Xiao Ming goes to the bowling alley for a game of bowling.

In a game of bowling, there are ten frames, each with ten pins. 1 point is awarded for every pin knocked down, and a maximum of 10 points can be awarded for every frame. A player can roll two times in each frame. If a player gets a strike in the 10th frame, then the player can roll one more time.

Xiao Ming has played eight frames, so he now has only two frames left in his game. Let's assume Xiao Ming did not score anything in the first 8 frames (if only there are bumpers...). Please calculate the total score earned for the remaining two frames.

Input instructions:

  1. Input the scores for each throw in the remaining two frames, one line for each throw.
  2. If all the pins are knocked out in the tenth frame, input the score for the extra throw in a new line.

Output instruction: Output the score earned from the two frames.

Sample Input Expected Output
2
5
7
1 		15
5
5
10
0
8 		28
