Lucky Draw Grid

Fall 2018: Question 32 (use Python to write this program)
Spring 2018: Question 5 (use Python to write this program)
Fall 2017: Question 33 (use C language to write this program)

Xiao Ming has a n-by-m (both n and m are from 1 to 10) lucky draw grid, and there are 1st to 9th prizes in the grid. His hand slipped, causing the lucky draw grid to move when he was walking on the stair.

The grid has 3 ways in which it can move:

  1. rotate 90 degrees to the left
  2. rotate 90 degrees to the right
  3. rotate 180 degrees to the right

Write a program to help Xiao Ming find the original arrangement of the prizes in the grid.

Input instructions:

  1. Input two numbers, separated by a space.
  2. For each line, input the corresponding amount of numbers, which represent how the prizes are arranged. Do not use a space in between numbers. The numbers can be repeated.
  3. Lastly, input how the grid has rotated. Input is as follows:
    • 1: rotate 90 degrees to the left
    • 2: rotate 90 degrees to the right
    • 3: rotate 180 degrees to the right

Output instructions:

  1. Output the starting arrangement of the prizes in the grid.
  2. If the input is invalid, output E.
