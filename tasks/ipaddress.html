IP Address

Spring 2018: Question 29 (use C language to write this program)

IP addresses consist of four numbers, with each number in the range of 0 to 255 and separated with a period, for example: 140.124.1.255.

To make an IP address easier to remember, we can add a nickname.

If the first two numbers in the IP address of multiple computers are the same, then it can be said that these computers are on the same local network.

Please write a program to input the IP address of several computers, and find out the computers that are on the same local network.

Input instructions:

  1. For every line of input, input the IP address and nickname of a computer, with the IP address and nickname separated with a comma.
  2. Input 0.0.0.0,none to end the inputs.

Output instructions:

  1. If there are computers that are on the same network, output the nicknames of the computers, in order of input, as follows:
    machines [nickname 1] and [nickname 2] are on the same local network.
  2. If there are n (n >= 3) computers on the same network, output the nicknames as follows:
    machines [nickname 1], [nickname 2], ... and [nickname n] are on the same local network.
  3. If the IP address input is invalid (any number greater than 255 or less than 0), output the nicknames of the erroneous IP addresses, in order of input, as follows:
    machine [nickname] is error ip
    Output only ONE nickname per line.

Sample Input Expected Output
111.22.3.44,blue
55.66.7.88,red
111.22.55.55,black
111.2.5.66,green
555.66.1.2,John
111.22.1.1,Tom
55.66.11.22,Mary
1111.1.1.1,Ken
0.0.0.0,none 		machines blue, black and Tom are on the same local network.
machines red and Mary are on the same local network.
machine John is error ip
machine Ken is error ip
