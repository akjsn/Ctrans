Plagiarism Checker

Spring 2018: Question 26 (use C language to write this program)

In order to check for any possible plagiarism in two different theses, we will need a tool for scanning a thesis.

Given two pieces of text, write a program to check which part of the sentence is most likely plagiarized.

Input instruction: Input two pieces of text, in two lines.

Output instructions:

  1. Output the longest part of the second sentence which is plagiarized in the first sentence.
  2. Output how similar is the plagiarized section when compared to the entire piece of the first text (Length of plagiarized section/Number of words in first text, without punctuation marks), in its simplest fraction form.
