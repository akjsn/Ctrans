Shape Of Stars

Fall 2018: Question 39 (use C language to make this program)
Fall 2018: Question 15 (use Python to make this program)
Fall 2017: Question 21 (use C language to make this program)

Write a program to output a shape using stars (*).

Remember to write a function to output the stars.
You should use for loop or while loop to output the stars.

Input instructions:

  1. First, input a number to determine the output shape: 1 for right-pointed triangle, 2 for left-pointed triangle, and 3 for diamond.
  2. Input a second number which will determine the number of rows the shape will span.
    This number has to be an odd number from 3 to 21.

Output instruction: Output the shape.
