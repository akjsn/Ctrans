Types of Triangles

Fall 2018: Question 6 (use Python to write this program)
Fall 2017: Question 6 (use C language to make this program)

If three lengths can form a triangle, please decide whether the triangle is an equilateral triangle, or isosceles triangle, or a scalene triangle.

You have to write a function int getTriangle(int a, int b, int c); in order to find the output.

Input instruction: Input three sides.

Output instructions:

  1. If the sum of the two shorter sides is less than the length of the longest side, or if any of the sides is 0 or less, then the three sides CANNOT form a triangle. Output “not triangle” if this happens.
  2. If the lengths of all three sides are equal, then the three sides will form an equilateral triangle. Output “equilateral triangle” if this happens.
  3. If the lengths of any two sides are equal, and the sum of the squares of two sides is more than the square of the third side, then the three sides will form an isosceles triangle. Output “isosceles triangle” if this happens.
  4. If none of the the above conditions are satisfied, then the three sides will form a scalene triangle. Output “triangle” if this happens.

Sample Input Expected Output
4 1 1 not triangle
3 3 3 equilateral triangle
3 2 3 isosceles triangle
7 8 9 triangle

Good Luck!
