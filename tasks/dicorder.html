In Dictionary Order

Spring 2019: Question 8 (use C language to write this program)

In a dictionary, words are arranged in alphabetical order from A, B, C, and all the way to Z.
If there are two words which have the same starting letter, the following letters are compared starting from the second letter.
If there are two different words which have the same beginning but are of different lengths (e.g. sigh and sight), the shorter word comes first.

Write a program to arrange a series of strings in alphabetical order.

Input instructions:

  1. First, input a word.
  2. Next, input a series of strings, one string at a time.
  3. Input -1 to end all input.

Output instructions: Remove the word from each string, and arrange the strings, minus the word, in alphabetical order.

Sample Input Expected Output
cat
pqwecatkoiashd
wecaqozeokqr
apejioscatvab
apejiosva
iqwcatepokpfed
-1 		apejiosva
apejiosvab
iqwepokpfed
pqwekoiashd
wecaqozeokqr
