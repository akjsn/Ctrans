Shape Structure II

Spring 2018: Question 32 (use C language to make this program)

Based on the following code, write a program to calculate the perimeter of various shapes, and find the total perimeter of all the shapes.

#include <stdio.h>
#include <string.h>

#define ShapeText(TYPE)
   char name[10];
   float (*perimeter)(struct TYPE*);

typedef struct _Shape {
   ShapeText(_Shape);
} Shape;

typedef struct _Circle {
   ShapeText(_Circle);
   float radius;
} Circle;

float ShapeArea(Shape *obj) { return 0; }
float ShapePerimeter(Shape *obj) { return 0; }

void ShapeNew(Shape *obj) {
   strcpy(obj->name,"shape");
   obj->perimeter = ShapePerimeter;
}

float CircleArea(Circle *obj) { return 3.14 * obj->radius * obj->radius; }
float CirclePerimeter(Circle *obj) { return 2*3.14 * obj->radius; }

void CircleNew(Circle *obj) {
   strcpy(obj->name,"circle");
   obj->perimeter = CirclePerimeter;
}

int main() {
   int i;
   Shape s;
   Circle c;
   ShapeNew(&s);
   CircleNew(&c);
    c.radius=3.0;
   Shape *list[] = { &s, (Shape*) &c};
    for (i=0; i<2; i++) {
      Shape *o = list[i];
      printf("%s.perimeter()=%G\n", o->name, o->perimeter(o));
   }
   system("PAUSE");
}
WARNING: YOU MUST REMOVE system("PAUSE"); FROM THE SAMPLE CODE WHEN WRITING YOUR PROGRAM, OTHERWISE YOUR PROGRAM WILL NOT BE ACCEPTED.

In this exercise, you must use struct to define the classes Circle, Rectangle, Square, and Triangle.

The member variables of the above classes are as follows:

  • Circle: radius
  • Rectangle: width, height
  • Square: side
  • Triangle: three sides -- a, b, c

Input instructions:

  1. First, input the number of shapes N.
  2. Next, for every line from line 2 to line N+1, input the shape, and the sizes of all the required elements of that shape, all in one line, and separated with a space.
    Input format is one of the following:
    • Circle: C radius (input 1 number)
    • Rectangle: R width height (input 2 numbers)
    • Square: S side (input 1 number)
    • Triangle: T a b c (input 3 numbers)

Output instructions:

  1. From line 1 to line N, output the perimeter of every shape.
    For the output of Circle, take π = 4.
  2. Lastly, output the total perimeter of all shapes.
  3. If the input is invalid, output “Error!”.

Sample Input Expected Output

T 3 4 5
S 1
R 2 3
C 0
C 1
T 3 4 6 		Error!
12
4
