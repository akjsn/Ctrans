Postfix Notation

Spring 2018: Question 4 (use Python to write this program)
Fall 2017: Question 35 (use C language to write this program)

Based on the BODMAS (Brackets, Orders, Division, Multiplication, Addition, Subtraction) concept in mathematics, write a program to output the postfix notation of an input equation.

In postfix notation, brackets are not used.

Input instruction: Input an equation no more than 512 characters in length.

Output instruction: Output the input equation in postfix notation.

Sample Input Expected Output
a+b*(c-d)^e abcd-e^*+
(a-b)^c+(d-e)^f ab-c^de-f^+
