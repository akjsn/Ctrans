Salary Raise

Spring 2018: Question 11 (use Python to write this program)

The boss of a company wants to raise the salary of his staff, and decides that a staff’s salary will be increased by 20% if the staff’s job performance and seniority both meet the company’s standards.
The boss wants to know, if he uses this method, then how much should he spend to raise the salary of his staff?

Input instructions:

  1. First, input the number of staff N. This number has to be between 2 and 100.
  2. Input N lines of data, which contain the staff’s ID and current salary, separated with a space. You can store this information in a list or in a dictionary.
  3. Input the IDs of staff whose job performance this year is satisfactory, with the IDs separated with a space. Please store this information in a set.
  4. Input the IDs of staff whose seniority this year is satisfactory, with the IDs separated with a space. Please store this information in a set.

Output instructions:

  1. Output the amount that the boss must spend to raise the salaries of the staff who qualify.
  2. If the first input (N) is invalid, output “Error”.

Sample Input Expected Output
5
001 1000
002 2000
003 500
004 1500
005 100
001 003 004 005
002 003 005 		120
