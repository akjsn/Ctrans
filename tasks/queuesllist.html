Queues Using Linked Lists

Note: This exercise uses multiple queues. For a single queue, click here.
Spring 2019: Question 28 (use C language to make this program)

Make a program to demonstrate the concept of queues using linked lists.

Designing queues in C require structures or classes, and these structures or classes are linked to one another using a linked list.
In this exercise, the contents of the queue will be numbers (int).

Functions:

  1. Create Queue
    Builds and names a new queue.
  2. Enqueue
    Input a queue name, and insert a number.
  3. Dequeue
    Input a queue name, and an item will be removed from the end of the queue.
    If there is nothing in the queue, output "Queue is empty".
  4. Merge Queue
    Input two queue names, then link the second queue to the first queue. The second queue will be deleted upon merge.
  5. Print All Queues
    Outputs all queue data, which contains the queue name, size, and elements.
    Output should be based on the following code:
    printf("****************************************");
    printf("Queue Name:[queue name] Queue Size:[queue size] Queue Element:[queue elements]", queue->name, count); // first queue
    printf("Queue Name:[queue name] Queue Size:[queue size] Queue Element:[queue elements]", queue->name, count); // second queue
    // and so on...
    printf("****************************************");
    If there are no queues, output "Isn't have any queue".
    If a queue exists, but has no elements, output "Queue is empty" after "Queue Element:".
  6. Exit
    Exits the program.

Input instruction: Input one of the following numbers to enter into a function.

  1. Create Queue
  2. Enqueue
  3. Dequeue
  4. Merge Queue
  5. Print All Queues
  6. Exit

Output instruction: For Enqueue, Dequeue, and Merge Queue functions, if the specified queue name does not exist, output "Queue [name of queue] isn't exist".
