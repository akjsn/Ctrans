Equation of A Straight Line

Fall 2018: Question 30 (use Python to write this program)
Spring 2018: Question 6 (use Python to write this program)
Fall 2017: Question 30 (use C language to write this program)

Write a function from which two points on the coordinate system, (x1, y1), (x2, y2), are input, and then output the equation of the line which passes through these two points using the following two methods:

  1. Output as y = mx + b

    m = (y1 - y2) / (x1 - x2)
    b = (x2y1 - x1y2) / (x2 - x1)

    If done in C: void f1_m(int x1, int y1, int x2, int y2, double m, double b);
    If done in Python: def f1_m(x1, y1, x2, y2, m, b):
  2. Output as y = (m1 / m2)x + (b1 / b2)
    m1 = (y1 - y2) m2 = (x1 - x2)
    b1 = (x2y1 - x1y2) b2 = (x2 - x1)
    If done in C: void f2(int x1, int y1, int x2, int y2, int m1, int m2, int b1, int b2);
    If done in Python: def f2(x1, y1, x2, y2, m1, m2, b1, b2);

Input instruction: Input four integers in order: x1, y1, x2, y2.

Output instructions:

  1. or y = mx + b, output the results correct to 2 decimal places.
  2. For y = m1 / m2x + b1 / b2, output the results as fractions. You do not have to simplify the fracions.
  3. The resulting equation may have either x or y missing.
    • If x is missing, output y = b.
    • If y is missing, output x = -b / m.
    • If m, b are integers, then express these two numbers as integers.
  4. If the input is invalid, output "Error!".

Sample Input Expected Output
3 4 -3 0 y=0.67x+2
y=4/6x+2
3.3 4 -3 0 Error!
1 0 0 1 y=-x+1
y=-x+1
