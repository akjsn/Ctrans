Checking Course Timings

Fall 2018: Question 11 (use Python to make this program)
Spring 2018: Question 7 (use Python to make this program)
Fall 2017: Question 11 (use C language to make this program)

Please check whether or not the timings of three courses clash.

Input instruction: Input the following in order:

  1. Course ID (four digits)
  2. Course hours per week (1 or 2 hours)
  3. Course timings (two digits, as follows):
    • First digit: 1 for Monday, 2 for Tuesday, 3 for Wednesday, 4 for Thursday, and 5 for Friday.
    • Second digit: A number from 1 to 8, indicating the period in which the course is held.

Output instructions:

  1. If the timings of the courses clash, please output the courses which clash in the following order: Course ID, Course ID, clashed timings
  2. If none of the courses’ timings clash, please output “correct”.
  3. If there are any errors in input, please output “-1”.
