Generating Waves

Spring 2018: Question 3 (use Python to write this program)

In this exercise, based on a given amplitude and frequency, you have to build a program to generate waves.

In postfix notation, brackets are not used.

Input instructions:

  1. First, input a number N. This number represents the number of test cases.
  2. For each test case, there will be 2 numbers input in two separate lines:
    • The first number represents the amplitude (A) of the wave (this number must be less than 9)
    • The second number represents the frequency (F).

Output instructions:

  1. For every test case, output a wave with amplitude A and frequency F.
  2. The wave output is formed using the number from the current height of the wave.
  3. Leave a blank line after every wave, and after every test case.
