Number Maze

Spring 2018: Question 16 (use Python to make this program)
Fall 2017: Question 42 (use C language to make this program)

Given a number maze in the form of a 2D array, consisting of numbers (from 0 to 9).
To get from the top left edge of the maze to the bottom right edge of the maze, you can move up, down, left, and right (but not diagonally).

Write a program to find out the lowest cost needed to go from the top left edge of the maze to the bottom right edge of the maze.

Input instruction: Input the following in order:

  1. For the first line, input the number of lines N (with the value of N being from 1 to 9).
  2. For the second line, input the number of columns M (with the value of M being from 1 to 9).
  3. Next, input N lines of M numbers, with each number separated by a space.

Output instruction: Output the lowest cost needed to go from the top left edge of the maze to the bottom right edge of the maze.

Sample Input Explanation Expected Output
4
5
0 3 1 2 9
7 3 4 9 9
1 7 5 5 3
2 3 4 2 5
0 3 1 2 9
7 3 4 9 9
1 7 5 5 3
2 3 4 2 5
The maze and the path taken to achieve the given result is shown in the above figure.
In this case, the lowest cost is 0 + 7 + 1 + 2 + 3 + 4 + 2 + 5 = 24. 		24
