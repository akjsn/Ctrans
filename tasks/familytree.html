Family Tree

Spring 2018: Question 28 (use C language to write this program)
Spring 2018: Question 17 (use Python to write this program)

Given a family tree, with a number of family members and the relationships between them.
From the family tree, it is known that:

  • 0 is the child of 7
  • 1, 2, and 3 are the children of 0
  • 4 and 5 are the children of 1
  • 6 is the child of 3

The furthest degree of relationship is from 4 (or 5) to 6 is 6, from which the distance is 4 (4→1, 1→0, 0→3, 3→6).

Based on the above family tree, please write a program to find out the degree of relationship between each and every family member. Assume that there is only one parent for every child.

Input instructions:

  1. First, input a number n (integer). This number represents the number of family members.
    Each family member is represented by a number from 0 to n-1.
  2. Next, input n-1 lines, with each line containing two integers a and b, separated by a space.
    These integers represent the relationship between two members, from which b is the child of a.

Output instruction: Output the degree of relationship from each and every member.
