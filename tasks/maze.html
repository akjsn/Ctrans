Maze

Spring 2019: Question 15 (use C language to write this program)

Given a maze of n-by-n size, with its starting point at (1, 1), and its finish point at (n-2, n-2), write a program to traverse the maze from start to finish.

Note that on an intersection, determine the route to be traversed in the following order: down, up, left, right.

Input instructions:

  1. First, input a number, which represents the size of the maze.
  2. Next, input n lines of n numbers each, with each number separated by a space. Input 0 for paths, and 1 for walls.

Output instruction: Output the path to be taken in order to go from start to finish. Output * for traversed paths, and # for the correct path.
