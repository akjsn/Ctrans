Spring 2019: Question 18 (use C language to write this program)

Write a program to simulate a digital circuit. Input is an 8-bit binary number (00000000 (0) to 11111111 (255) in range), and output will be in the form of an 11-bit binary number.

The digital circuit has three components:

  1. Loop circuit
    The loop circuit does operations on a decimal number m as follows:
    • C(m) = m if m = 0 or m = 1
    • C(m) = C(m / 2) if m is even
    • C(m) = C((m + 1) / 2) if m is odd
  2. Counter
    The counter tracks how many times the number runs through the loop circuit. For example:
    • m=00001010 (decimal 10), the loop circuit will return 5, 3, 2, and 1 in its iterations.
      C(10) -> C(5) -> C(3) -> C(2) -> C(1) -> 1. The number ran the loop circuit 4 times. Therefore, R(10) = 4.
    • m=00000011 (decimal 3), the loop circuit will return 2 and 1 in its iterations.
      C(3) -> C(2) -> C(1) -> 1. The number ran the loop circuit 2 times. Therefore, R(10) = 2.
  3. Recorder
    The recorder calculates the total loop circuit runs from m = 1, m = 2, m = 3, ..., m = n.
    For example, if user inputs n = 3, then the recorder calculates the sum of R(0), R(1), R(2), and R(3).
    R(0) = 0, R(1) = 0, R(2) = 1, R(3) = 2; therefore, the output from the recorder is 0 + 0 + 1 + 2 = 3.
    Note that this output should be converted to 11-bit binary; therefore from the above example, the output will be 00000000011.

Input instructions:

  1. Input an 8-bit binary number.
  2. After that, input 0 to continue input (go back to Step 1), or input -1 to end input.

Output instruction: Output the 11-bit binary number result generated from the recorder output.
