Shape Structure (Linux Terminal)

Spring 2019: Question 25 (use C language to make this program)

Based on the earlier Linux Command Line Interface Simulator, use struct to define the structures Shape, Circle, Rectangle, Square, and Triangle.

The member variables of the above classes are as follows:

  • Circle: radius
  • Rectangle: width, height
  • Square: side
  • Triangle: three sides -- a, b, c

You are required to use the following code to write the following program:

#include
#include

#define ShapeText(TYPE)
char name[10];
float (*perimeter)(struct TYPE*);

typedef struct _Shape { // Shape object, no fields
    ShapeText(_Shape);
} Shape;

typedef struct _Circle {
    ShapeText(_Circle);
    float radius;
} Circle;

float ShapeArea(Shape *obj) { return 0; }
float ShapePerimeter(Shape *obj) { return 0; }

void ShapeNew(Shape *obj) {
    strcpy(obj->name,"shape");
    obj->perimeter = ShapePerimeter;
}

float CircleArea(Circle *obj) { return 3.14 * obj->radius * obj->radius; }
float CirclePerimeter(Circle *obj) { return 2*3.14 * obj->radius; }

void CircleNew(Circle *obj) {
    strcpy(obj->name,"circle");
    obj->perimeter = CirclePerimeter;
}

int main() {
    int i;
    Shape s;
    Circle c;
    ShapeNew(&s);
    CircleNew(&c);
    c.radius=3.0;
    Shape *list[] = { &s, (Shape*) &c};
    for (i=0; i<2; i++) {
        Shape *o = list[i];
        printf("%s.perimeter()=%G\n", o->name, o->perimeter(o)); //multi-typed
    }
}

Input instructions:

  1. Input uses the following format: touch [shape_code][name(parameters]
    Use commas to separate the values in each parameter.
    Examples:
    • touch R r_1(2,3)
      Makes a Rectangle shape named r_1 with width 2 and height 3.
    • touch C c_1(1) R r_1(2,3) S s_1(1) T t_1(3,4,5)
      Makes four shapes: Circle named c_1 with radius 1, Rectangle named r_1 with width 2 and height 3, Square named s_1 with side 1, and Triangle named t_1 with three sides 3, 4, and 5.
  2. Shape code is as follows:
    • Circle: C (input 1 number)
    • Rectangle: R (input 2 numbers)
    • Square: S (input 1 number)
    • Triangle: T (input 3 numbers)
  3. Type exit to exit the program.

Output instructions:

  1. Output by inputting the command cat.
    Output the files in alphabetical order, unless stated.
    Examples:
    • cat *: output from all files, with filename in alphabetical order. cat *file: output from all files which end with file cat fi*e: output from all files which begin with fi and end with e cat file*: output from all files which begin with file cat file1 file2 output from file1, then from file2.
  2. Output the area and perimeter of the shain one line, separated with a comma.
