Spring 2019: Question 20 (use C language to write this program)

Two people are playing bingo. Each player has an N-by-N grid (1 < N < 1000) containg numbers N1, N2, ... , N9.

Note that if Ni < Nj when i < j and Ni > Nj when i > j, then i <= Ni, Nj <= 9, 1 <= i, and j <= 9.

Three numbers C1, C2, and C3 are then chosen. Note that if Ci <> Cj when i <> j, then i <= Ci, Cj <= 3, 1 <= i, and j <= 3.

There are three outcomes from this game: Player 1 wins, Player 2 wins, or the match ended in a draw. If the three numbers form a straight line in a player’s bingo grid horizontally, vertially, or diagonally, then the player wins.

Input instruction: Input four lines:

  1. First, input a number N, which represents the size of the grid (1 < N < 1000).
  2. Then, input two lines of N*N numbers, with each number separated by a space.
    The first line is the bingo card of Player 1, and the second line is the bingo card of Player 2.
  3. The numbers for inputs in (2) and (3) must be an integer in the range of 1-9, and all numbers must be different.
  4. Finally, input M numbers (also separated with a space). These numbers represent the picked numbers. (1 < M < N * N)

Output instructions:

  1. First, output the outcome of the game:
    • If Player 1 wins, output “Player1 wins x, Player2 loses y”.
    • If Player 2 wins, output “Player2 wins x, Player1 loses y”.
    • If both players win or no one wins, output “Draw x”.
  2. For x and y, output the number of lines the respective player has. In case of a draw, output the number of lines each player has.

Sample Input Expected Output
3
1 2 3 4 5 6 7 8 9
9 8 7 6 5 4 3 2 1
1 2 3 		Draw 1
3
1 4 2 5 6 7 3 8 9
9 8 4 7 6 3 5 2 1
2 7 9 		Player1 wins 1, Player2 loses 0
3
9 4 2 5 6 1 3 8 7
9 8 4 3 6 7 5 2 1
9 4 8 		Player2 wins 1, Player1 loses 0
