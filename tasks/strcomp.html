String Comparison

Spring 2019: Question 9 (use C language to write this program)

Write a program to compare two strings, and find the longest shared section of the two strings.

Input instruction: Input two strings.

Output instructions: Output the longest shared section of the two strings.

Test cases will not include two shared string sections which have the same length.
Sample Input Expected Output
qqwweee
qqwwweee 		wweee
