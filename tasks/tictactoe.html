Tic-Tac-Toe

Spring 2018: Question 24 (use Python to write this program)

Tic-tac-toe is a two-player game which uses a 3-by-3 grid, with positions as shown:

Input instructions:

  1. Input i to input the positions where the players make their mark.
    Input two lines of coordinates (separated with a space); the first line is the input for Player 1, and the second line is the input for Player 2.
  2. Input w to output the possible winning spaces for Player 1.
  3. Input d to output the possible winning spaces for all players.

Output instructions:

  1. If, after inputting i, any of the inputs are invalid (e.g. same space for Player 1 and Player 2), output “input error”, and everything that has been input after i up to this point are not considered.
  2. If w or d is input, output the result(s) as follows:
    • for Player 1: Player 1 will win with (x, y)
    • for Player 2: Player 2 will win with (x, y)
  3. Be sure to check if there is a winning player after inputs are made.
    • If Player 1 wins, output “Player 1 win”.
    • If Player 2 wins, output “Player 2 win”.
    • If the game ends with a tie (both players win or no one wins), output “Fair”.
