Stock Prices

Spring 2018: Question 23 (use Python to make this program)

Given an array of stock prices of a company every day during a period of time (30 days at most), write a program to find out the best price to buy and sell the company’s stocks.

Remember the rule: buy low, sell high – that’s how you earn a fortune in stock market trading!

Note that you CANNOT buy and sell a company’s stocks on the same day.

Input instructions:

  1. First input the number of days n. (n <= 30)
  2. Input the stock prices for each day during the period of n days. Separate each piece of data with a space.

Output instruction: Output the most favorable buying and selling price.

Please use recursion in this exercise.
Sample Input Expected Output
7
55.41 109.22 48.27 81.58 105.54 94.49 12.25 		48.27
105.54
