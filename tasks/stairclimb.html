Climbing The Stairs

Spring 2017: Question 17 (use C language to write this program)

Xiao Ying wants to climb a mountain, and he practices by climbing the stairs of Taipei 101.

Xiao Ying can climb at most three steps at a time (i.e. he can climb 1, 2, or 3 steps at one time); therefore, there are many possible ways to climb the stairs.

For example, if Xiao Ying wants to climb 3 steps, there are four ways:

  • climb one step at a time
  • climb one step, then climb 2 steps
  • climb 2 steps, then climb 1 step
  • climb 3 steps
If Xiao Ying wants to climb 4 steps, then there are 7 ways.

Xiao Ying can climb at most 70 steps.

Hints

  1. The total number of possible ways to climb n steps is the sum of the total number of possible ways to climb n-1, n-2, and n-3 steps.
    f(n) = f(n-1) + f(n-2) + f(n-3)
  2. Please consider whether to use the recursive or non-recursive method to solve this problem. You may also use pointers.
  3. Please use unsigned long long int x; printf("%lld", x);

WARNING

Your program must complete execution within 8 seconds, otherwise your program will not be accepted !!!

Input instruction: Input the number of steps to be climbed.

Output instruction: Output the number of possible ways to climb the input number of steps.

Sample Input Expected Output
4 17
12 927
67 333269972246340068
