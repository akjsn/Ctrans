Visiting a Computer Fair

Spring 2019: Question 29 (use C language to make this program)

James is going to a computer fair at the end of this month. This computer fair will be a large one, and there will be many exhibitors in the fair. James would like to know the locations of the booth he wants to visit in the fair.

In this exercise, we will be using linked lists to perform sort operations.
The following is the code used to define the Company structure.

typedef struct Company{
    char name[10];    // company name
    char site[10];    // booth location
    char topic[10];   // topic/product category (e.g. AI„ÄÅBlockchain...)
    char country[10]; // country
    int award;        // awards won
}

Input instructions:

  1. First, input a series of exhibitor data, one line at a time, with each data in the following format: name, site, topic, country, award.
  2. Input -1 to end the abovementioned input and enter search mode.
  3. In search mode, input consists of one of the exhibitor identifiers (name, site, topic, country, award) and an integer (1 (descending) or 2 (ascending)).
  4. Input exit to end the program.

Output instruction: In search mode, output all the exhibitor data sorted from on the parameters given in input. The exhibitor data should be sorted based on its ASCII characters.

  1. If two or more exhibitor data have the exact same value, then the output order for these data will be based on its booth site.
  2. Note that the five pieces of exhibitor information are in the form of a continuous string, and special characters are included.
  3. Leave a line after each query.
