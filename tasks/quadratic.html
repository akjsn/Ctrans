Solving A Quadratic Equation

Fall 2018: Question 2 (use Python to write this program)
Fall 2017: Question 2 (use C language to make this program)

Calculate the real roots of the quadratic equation Ax2 + Bx + C = 0.

If done in C, you have to use #include<math.h> to write this program.

Input instruction: three numbers (all int): A, B, and C

Output instructions:

  1. First root (in double): x1 = ((-b) + sqrt(b * b - 4 * a * c)) / (2 * a)
    Second root (in double): x2 = ((-b) - sqrt(b * b - 4 * a * c)) / (2 * a)
  2. Display the roots in ONE decimal place, for example: print("%.1f" %x1);

Sample Input Expected Output
1
-2
1 		1.0
1.0
