Linux Command Line Interface Simulator

Spring 2019: Question 24 (use C language to make this program)

Write a program to simulate the Linux Command Line Interface.
The file structure is as follows, and the functions are explained below.

struct file {
char* name;
char c[];
};

Functions:

  1. Write File
    Command: touch file
    Examples:
    touch file
    touch file1 file2 file3
  2. Input text file (this will do an edit operation to a file)
    Command: vim file
    vim is one of the many text editors included in many UNIX systems, including Linux. Click here for more information.
    Example: (Note: the a in Line 2 of this example signifies the start of file writing and does not need to be written into the file)
    vim file
    a
    123
    456
    789
    Commands in vim:
    • :w (saves)
    • :q (exit without saving, output Error and return to command terminal)
    • :q! (force exit without saving)
    • :wq (save and exit)
  3. Read File
    Commands:
    • cat file (reads file)
    • head file (reads file from the beginning)
    • tail file (reads file from the end)
    Sample Input Sample Output
    cat file 123
    456
    789
    cat *file (Output files in alphabetical order, and output the contents)
    cat file* (Output files in alphabetical order, and output the contents)
    Sample Input Explanation Sample Output
    head file Initially set to first 10 lines 123
    456
    789
    head file 2 Read first 2 lines 123
    456
    tail file Initially set to last 10 lines 123
    456
    789
    tail file 2 Read last 2 lines 456
    789
  4. Delete File
    Command: rm file
    Examples:
    rm file
    rm file1 file2
  5. List Files
    Command: ls
    Sample Input Sample Output
    ls file1 file2 file3
  6. Exit
    Command: exit

Sample Input Expected Output
touch file1 file2
vim file1
a
1
2
3
:wq
vim file2
abcd
:wq
cat file*
exit 		1
2
3
bcd
