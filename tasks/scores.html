Calculating the Total Score and Average Score

Fall 2018: Question 1 (use Python to write this program)
Fall 2017: Question 1 (use C language to write this program)

A student takes these three courses: Chinese, Introduction To Computing, and Computer Programming.

From the following inputs: name (one char), ID (int), and the scores of three subjects (int), do the following:

  1. Calculate the total score and average score.
  2. Display the name, student ID, total score, and average score.
