Overlapping Lines

Fall 2018: Question 12 (use Python to make this program)
Fall 2017: Question 12 (use C language to make this program)

Given the starting and ending points of three line sections on an axis, please write a program to find out the total length of the three sections.

Remember that overlapping sections can only be counted once.

Input instruction: Input the start and end points of three lines.

Output instruction: Output the total length of the three sections when placed on top of one another.

Sample Input Explanation Expected Output
-1 1 0 2 1 3 4
