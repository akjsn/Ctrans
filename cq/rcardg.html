Report Card Grade

Every time the report card is out, you will get to know how well you did in the subjects you take.

You would like to know the term grade and GPA for the courses you take. Write a program to achieve this function.

Input instructions:

  1. For each line, input two numbers in order: score (integer between -1 to 100), credit count (positive integer). Separate each of the two numbers with a space.
    Note that a score of -1 represents the score of a course that was withdrawn, and this course's score will not count towards the term grade and GPA.
  2. An input of -1 -1 ends all input.

Output instructions:

  1. In the first line, output the total credits taken during the semester, excluding credits for courses that have been withdrawn.
  2. In the second line, output the total credits earned during the semester. To earn credits for a course, the course's score must be at least 60.

  4. In the fourth line, output the GPA correct to 2 decimal places. The GPA is calculated as follows:
    To determine the grade points, use the following table:
    Score Grade Points
    90-100 4
    80-89 3
    70-79 2
    60-69 1
    0-59 0
  5. Lastly, if all the scores are at least 60, and there are no withdrawn courses, output "You passed all the subjects for this semester!"
  6. If any of the inputs is invalid, output "ERROR" upon detection, and terminate the program.
