Indian Numerals

based on UVA10101: Bangla Numbers

In Indian English, lakh is used to represent 100,000, and crore is used to represent 10 million. 1,000 is still represented in the usual form in English language.

In addition, a number in Indian figures uses the following pattern on the placement of the comma separator, starting from the ones place:

  1. One comma after three numbers
  2. One comma after two numbers
  3. One comma after two numbers
If there are more numbers after the above pattern is completed, the pattern starts again.

Examples of numbers in Indian numerals:

  • 100,000 is written as 1,00,000
  • 10,000,000 is written as 1,00,00,000
  • 1,000,000,000 is written as 100,00,00,000

Write a program from which the user inputs a number (unsigned long int) and the program should convert the input number into Indian numerals, both in number and text.

Input instruction: Input a number.

Output instruction: Output the following in order: Indian numeral (in numbers), Indian numeral (in text).
