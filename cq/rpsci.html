Rock-Paper-Scissors

You must be familiar with this classic hand game of rock-paper-scissors.

Write a program to simulate a game of rock-paper-scissors between two players.

Your program should allow continuous input.
Your program should also be able to track the scores of the two players.

Input instructions:

  1. First, input a number n. This number represents the points needed to win the game.
  2. Next, for each line, input two uppercase characters representing the moves by Player 1 and Player 2, separated with a space.
    Input R for rock, P for paper, and S for scissors.

Output instructions:

  1. As the game goes on, output who wins the round.
    Output "Player 1 wins" if Player 1 wins; or "Player 2 wins" if Player 2 wins; and the winning player gets a point.
    If there is a draw, output "Draw" (neither player earns points if a draw happened).
  2. If there are characters outside the abovementioned permitted characters, output "Error".
    Neither player earns a point in that round.
  3. In either case, output the current score of both players.
  4. Once a player reaches the points needed to win the game, output "Player 1 has won the game!" if Player 1 wins; or "Player 2 has won the game!" if Player 2 wins.

Sample Input Expected Output
3
R R
P S
R S
S P
S X
R P
P S 		Draw -- Player 1: 0, Player 2: 0
Player 2 wins -- Player 1: 0, Player 2: 1
Player 1 wins -- Player 1: 1, Player 2: 1
Player 1 wins -- Player 1: 2, Player 2: 1
Error -- Player 1: 2, Player 2: 1
Player 2 wins -- Player 1: 2, Player 2: 2
Player 1 wins -- Player 1: 3, Player 2: 2
Player 1 has won the game!
