Grid Painter

In image editing software, the paint bucket tool will color all areas of an image based on color similarity. The color similarity is determined pixel-by pixel.

In this exercise, we will use a 2-D array of characters to represent the image.
For the paint bucket tool, what we will do is replacing a character somewhere in the array with another character. If there are adjacent characters that are similar with the character to be replaced, that character is replaced as well.

Input instructions:

  1. First, input two numbers m and n, representing the size of the array.
  2. For the next m lines, input the contents of the array one line at a time. Each line consists of n characters.
  3. Input two numbers a and b (0 < a < m and 0 < a < n), which represents the location of the character to be replaced.
  4. Input one character to replace the character (and adjacent characters) in [a][b].

Output instructions:

  1. Output the modified array of characters.
  2. If there are any errors in the input, output ERROR upon detection and terminate the program.
